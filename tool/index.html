一言 ​

网址：hitokoto

用法：

  1. 这是一个api，你的请求应包含 do 和 c。

  2. do 应为 json(返回json) 或 text(返回文本)。

  3. c 应为以下参数之一(或留空)：

    参数说明
    a动画
    b漫画
    c游戏
    d文学
    e原创
    f来自网络
    g其他
    h影视
    i诗词
    j网易云
    k哲学
    l抖机灵

示例：

  • 返回类型为 文学 的 文本 ：https://hitokoto.lhf.us.kg/?do=text&c=d
  • 返回一段 json：https://hitokoto.lhf.us.kg/?do=json&c=all
